# API Reference

## Core Modules

### MuseeCore
- `StableID`: Cryptographically secure identifiers
- `PartialDate`: Flexible date representation
- `MuseeError`: Error handling types

### MuseeDomain
- `Person`: Individual entity with metadata
- `MediaAsset`: Media file representation
- `BiographicalClaim`: Attributed assertions
- `Tag`: Classification tags

### MuseeVision
- `VisionProcessor`: AI vision analysis
- `EROSSCalculator`: Beauty scoring
- `BodyMeasurementEstimator`: Physical measurements

### MuseeSearch
- `FacetedSearchQuery`: Search parameters
- `SearchEngine`: Search protocol
- `InMemorySearchEngine`: Basic implementation

### MuseeBundle
- `MuseeBundle`: .musee file handling
- `MuseeBundleSpec`: Bundle format constants

### MuseeMuseum
- `MuseumLibrary`: Museum management
- `MuseumSpec`: Museum structure constants

### MuseeMetadata
- Metadata extraction utilities
- C2PA verification
- EXIF/IPTC/XMP processing

### MuseeCLI
- Command-line interface
- Museum operations
