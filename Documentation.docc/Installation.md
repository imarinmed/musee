# Installation

## Requirements

- macOS 13.0 or later
- Xcode 14.0 or later
- Swift 5.7 or later

## Adding to Your Project

Add Musee as a dependency in your `Package.swift`:

```swift
dependencies: [
    .package(url: "https://github.com/yourorg/musee.git", from: "1.0.0")
]
```

Or in Xcode, add the package via File > Add Packages.
